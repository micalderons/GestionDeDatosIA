# Documentación del Proyecto - Gestión de Datos Entorno IA

## 1. Descripción
Este proyecto consiste en el desarrollo de una API utilizando FastAPI, desplegada en la nube mediante Render, aplicando contenedorización con Docker y automatización con GitHub Actions.

## 2. Tecnologías utilizadas
- Python 3.9
- FastAPI
- Docker
- GitHub
- GitHub Actions
- Render

## 3. Estructura del proyecto
El proyecto está organizado en carpetas separando código, configuración y documentación.

## 4. Contenedorización
Se utilizó Docker para asegurar que la aplicación funcione en cualquier entorno de forma consistente.

## 5. Automatización CI/CD
Se configuró GitHub Actions para validar automáticamente el código en cada push al repositorio.

## 6. Despliegue en la nube
La aplicación fue desplegada en Render como Web Service utilizando Docker.

## 7. URL de la aplicación
https://gestion-datos-api.onrender.com
