# Gestión de Datos - Entorno IA

## 1. Descripción del proyecto
Este proyecto consiste en una API básica desarrollada con FastAPI, preparada para ejecutarse localmente, contenerizarse con Docker y desplegarse en la nube mediante Render.

## 2. Objetivo
Configurar un entorno técnico en la nube aplicando principios de:
- contenedorización
- automatización
- trabajo colaborativo
- despliegue cloud

## 3. Tecnologías utilizadas
- Visual Studio Code
- Python 3.9
- FastAPI
- Uvicorn
- Git y GitHub
- GitHub Actions
- Docker
- Render
